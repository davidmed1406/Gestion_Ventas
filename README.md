# Gestion_Ventas
Análisis de ventas por medio de Excel, Power BI y MySQL

#Objetivo
Analizar datos para identificar tendencias , productos estrella, clientes 8020 y oportunidades de negocio

#Herramientas usadas
Excel, SQL y Power BI

#Insight
- Los principales Estados donde se concentran las ventas son Nueva York, California y Texas, siendo la Región Oesta la que tiene mayores ventas
- El promedio general de tiempo entre la orden del pedido y el envío del pedido es de 4 días, siendo Dakota del Norta el Estado con mejor respuesta y Distrito de Columbia el que tiene mayor atraso en los tiempos de envío
- En total son 796 clientes de los cuales se concentra el 80% de las ventas en 393 clientes siendo casi el 50% de los clientes

#Estructura
Analisis exploratorio con Excel Analisis y busquedas con SQL Dasboard a nivel Directivo, Gerencial y Operativo (no todos los puestos y jerarquias ocupan ver la misma información)
